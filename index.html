Navegadores web

Un navegador web (en ingles, web browser) es un software, aplicación o programa que permite el acceso a la Web, interpretando la información de distintos tipos de archivos y sitios web para que estos puedan ser vistos

La funcionalidad básica de un navegador web es permitir la visualizacion de documentos de texto, posiblemente con recursos multimedia incrustados

Ejemplos de navegadores web

Safari:Safari es un navegador web de codigo cerrado desarrollado por Apple Inc. Está disponible para macOS, iPasOS e iOS

Mozilla Firefox es un navegador web libre y de codigo abierto desarrollado para Linux, Android, iOS, macOS, y Microsoft Windows corrdinado por la corporacion Mozilla y la Fundación Mozilla

Google Chrome Google Chrome es un navegador web de codigo cerrado desarrollado por Google, aunque derivado de proyectos de código abierto
